# Tune3D — Airsoft Configurator

## Опис
Застосунок для налаштування airsoft зброї у 3D форматі. 
Дозволяє підбирати аксесуари (приціл, руків'я, приклад, лазер тощо) 
та переглядати їх у реальному часі на 3D моделі.

## Стек технологій
- HTML / CSS / JavaScript
- Three.js (3D рендеринг)
- GLTFLoader (завантаження 3D моделей)

## Запуск
Відкрити 3d_tuning.html у браузері
